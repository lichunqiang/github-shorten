github-shorten [![Version][version]](https://npmjs.org/package/github-shorten)
---------------------------

> Export your long github url to shorten.

## Install

```sh
$ npm install github-shorten -g
```

## Usage

```sh
$ github-shorten user/repos
```

* or

```sh
$ github-shorten -u user -r repos
```

* More

```sh
$ github-shorten --help
```
